# Usage

The main notebook is Final.ipynb. It leverages TensorFlow and takes advantage of a GPU if one is detected. In order to run the notebook, the dataset_4composers.zip file should be extracted and its content should be placed in a new build folder at the root of the project as follows:

```
build/
└── datasets/
        └── dataset_4composers/
            ├── Bach/
            ├── Beethoven/
            ├── Chopin/
            └── Mozart/
```
